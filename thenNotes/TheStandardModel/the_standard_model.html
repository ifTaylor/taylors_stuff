  1. Nucleon (Baryon)

    The particles that make up nucleus.

    1. Proton

      Charge - (+)1

      Quark characteristics: Up + Up + Down: (+)2/3 + (+)2/3 + (-)1/3 = +1

    2. Neutron

      Charge - 0

      Quark characteristics: Up + Down + Down: (+)2/3 + (-)1/3 + (-)1/3 = 0

    3. Quark Configurations

      Config Charges Nomenclature
      UUU 2/3 + 2/3 + 2/3 = 2 (Delta++(Delta baryons or resonances))
      UUD 2/3 + 2/3 - 1.3 = 1 (Proton)(Delta+)
      UDD 2/3 - 1/3 - 1/3 = 0 (Neutron)(Delta0)
      DDD -1/3 - 1/3 - 1/3 = -1 (Delta-)
      UDS 2/3 - 1/3 - 1/3 = 0 (Hyperon Lambda0)
      USS 2/3 - 1/3 - 1/3 = 0 (Hyperon Xi0)
      UUS 2/3 +2/3 -1/3 = 1 (Hyperon Sigma+)
      DDS -1/3 - 1/3 - 1/3 = -1 (Hyperon Sigma-)
      DSS -1/3 - 1/3 - 1/3 = -1 (Hyperon Xi-)
      SSS -1/3 - 1/3 - 1/3 = -1 (Hyperon Omega-)
